
## 📌 Description
Fraud Risk Monitoring system built using Python to detect suspicious transactions. Applied Z-score, IQR, and geo-risk analysis to flag anomalies. Generated risk scores and identified high-risk users.

## 🛠️ Technologies
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
